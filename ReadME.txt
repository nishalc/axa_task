This repo contains work conducted on a depression dataset from kaggle https://www.kaggle.com/datasets/anthonytherrien/depression-dataset/data.

Depression dataset analysis.ipynb - Jupyter notebook which contains exploratory analysis and model development
scr/model_train.py - script that can be ran to train a randomforest classifier, printing to console test set prediction performance
depression_data.csv - Kaggle dataset on mental health
depression_dataset_presentation - presentation on EDA and model development
requirements.txt - required modules
AXA_task_questions.docx - answers to the brief specific questions
